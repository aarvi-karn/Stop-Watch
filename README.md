# ⏱️ Stop Watch Web Application

A simple and responsive stopwatch web application built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features
- Start the stopwatch
- Pause/Stop the timer
- Reset functionality
- Clean and responsive UI

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
